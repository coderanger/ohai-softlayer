# Ohai-Softlayer Cookbook

Installs Ohai plugins for SoftLayer cloud metadata.

* `softlayer` – Ingests metadata from the SoftLayer API.
* `cloud_hack` – Add SoftLayer data to the `cloud` and `cloud_v2` hashes.
